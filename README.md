# Digital Trust Score ASN

Platform pengukuran dan monitoring kesadaran keamanan data pemerintah berbasis perilaku ASN untuk memperkuat ketahanan siber dan meningkatkan kepercayaan publik.

## Problem

Kesadaran keamanan ASN masih memiliki gap antara pengetahuan dan perilaku, meningkatkan risiko kebocoran data, serangan siber, dan fraud pada layanan publik serta sistem keuangan digital.

## Solution

Digital Trust Score ASN mengukur, menganalisis, dan memonitor perilaku keamanan ASN menggunakan model HAIS-Q, data analytics, dan machine learning, kemudian menampilkan dashboard risiko dan rekomendasi intervensi secara real-time.

## Features (MVP)

* Assessment berbasis K-A-B (HAIS-Q)
* Digital Trust Score
* Risk classification (Low–Medium–High)
* Dashboard monitoring real-time
* Rekomendasi intervensi otomatis

## Tech Stack

* Web Dashboard
* Python Backend
* Data Analytics
* Machine Learning
* REST API

## Impact

Meningkatkan kesadaran keamanan ASN, menurunkan risiko kebocoran data, serta memperkuat ketahanan siber pemerintah.
