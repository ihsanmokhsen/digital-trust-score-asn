# Digital Trust Score ASN

Prototype aplikasi web untuk monitoring kesadaran keamanan informasi ASN.

## Stack

- Next.js App Router
- React
- CSS custom

## Fitur Utama

- Login prototype untuk Admin, Pimpinan, dan ASN
- Dashboard ringkasan risiko per instansi
- Assessment perilaku keamanan informasi
- Mode test ASN dengan timer per soal
- Halaman nilai pribadi ASN
- Data dummy lokal tanpa API eksternal

## Akun Demo

- Admin: `admin / admin123`
- Pimpinan: `pimpinan / pimpinan123`
- ASN: `asn / asn123`

## Menjalankan Aplikasi

```bash
npm install
npm run dev
```

Lalu buka `http://localhost:3000`.
